百度杯十一月赛题CrackMe01

1

用IDA打开，只有一个信息框，所以从弹窗下手

搜索message API

看到MessageBoxW

双击进去，Ctrl+X查看调用，发现没有被调用，于是放过

弹窗有可能为窗口类，搜索window

找到ShowWindow


双击进入


自定义消息的位置

004028A5处 ebx累加

查看发送消息的API

Post MessageW

进入

F5查看伪码


把eax里的数组取出2c的长度，异或即可
